# Writing Style Guide — แค่เปลี่ยนคำ ก็ทำเงิน

> Source: Notion "Writing Style Guide — แค่เปลี่ยนคำ ก็ทำเงิน"
> Apply to all user-facing copy in the landing page

---

## 1. Tone หลัก

**เป็นกันเอง + ตรงไปตรงมา + ภาษาคนจริง**
เหมือนพี่สอนน้อง ไม่ใช่อาจารย์สอนนักศึกษา

❌ "ปัจจัยที่ส่งผลต่อการตัดสินใจซื้อของผู้บริโภค"
✅ "ทำไมลูกค้าอ่านจบแล้วไม่ซื้อ?"

❌ "วิธีการที่มีประสิทธิภาพในการเพิ่มยอดขาย"
✅ "เปลี่ยนแค่ประโยคเดียว ยอดขึ้น 3 เท่า"

❌ "หลักสูตรนี้ออกแบบมาเพื่อ..."
✅ "เล่มนี้เขียนให้คนที่..."

## 2. คำที่ต้องเปลี่ยน (global rules)

| ❌ ห้ามใช้ | ✅ ใช้แทน |
|---|---|
| คอร์ส | บทเรียน |
| ผู้เรียน | คุณ |
| หลักสูตร | เล่มนี้ / บทเรียนนี้ |
| นักเรียน | ลูกค้า / คนที่อยากขายเก่ง |
| ผลิตภัณฑ์ | สินค้า / ของที่ขาย |
| ดีมาก | + ตัวเลขเฉพาะ (เช่น "ลด 47%") |
| คุณภาพสูง | บอกผลลัพธ์ตรง ๆ |
| มีประสิทธิภาพ | ใช้ได้ผลจริง / ขายออก |

---

## 3. โครง headline สูตร

**Pattern A — ปฏิเสธความเชื่อเดิม**
> "ปัญหาไม่ใช่ X — ปัญหาคือ Y"
ตัวอย่าง: "ปัญหาไม่ใช่สินค้า — ปัญหาคือคำที่ใช้"

**Pattern B — เปรียบเทียบเจ็บ ๆ**
> "[Action ทุกวัน] ทำไมยังไม่[ผลที่หวัง]?"
ตัวอย่าง: "โพสต์ทุกวัน ยิงแอดทุกคืน ทำไมยังไม่มีคนกดซื้อ?"

**Pattern C — Mirror ความรู้สึก**
> "คุณเคย[รู้สึกแบบนี้]ไหม?"

---

## 4. ตัวเลขเฉพาะเจาะจง (เสมอ)

❌ "หลายบท"  → ✅ "24 บท"
❌ "หลากหลาย"  → ✅ "5 หมวด"
❌ "เพิ่มขึ้น"  → ✅ "เพิ่ม 2-5 เท่า"
❌ "ใช้เวลาไม่นาน"  → ✅ "10-15 นาที/บท"

---

## 5. Anti-AI tells (ลบทุกครั้งที่เจอ)

❌ "อย่างไรก็ตาม", "ดังนั้น", "นอกจากนี้"
❌ "ในยุคปัจจุบัน", "ในโลกธุรกิจ"
❌ ประโยคยาวซ้อน 3 ชั้น
❌ "เพื่อให้คุณสามารถ..."

✅ ตัดให้สั้น ใช้ break line, bullet, single-clause sentences

---

## 6. CTA wording

❌ "สมัครเรียนเลย" / "ลงทะเบียน" / "ซื้อทันที"
✅ "แอด LINE รับ 3 บทฟรี" / "ลองอ่านก่อน" / "กดที่นี่"

---

## 7. Trust building (no hard sell)

✅ "ถ้าไม่ใช่ก็ไม่เป็นไร"
✅ "อ่านก่อน ถ้าใช่ค่อยตัดสินใจ"
✅ "คืนเงินไม่ถามเหตุผล"

❌ "โอกาสสุดท้าย" / "เหลือไม่กี่ที่" / "ราคานี้วันนี้เท่านั้น" (ไม่ใช้ urgency เทียม)
